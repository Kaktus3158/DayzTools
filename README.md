# 🎮 DayZ Tools

Набор браузерных инструментов для DayZ-моддинга и редактирования конфигурационных файлов.

## 🌐 Онлайн-версия

Открыть панель инструментов:

[➡ DayZ Tools Control Panel](https://kaktus3158.github.io/DayzTools/)

---

## 🧰 Состав набора

### 📋 Types Editor
Редактор `types.xml` для настройки экономики, параметров лута и массового балансирования значений DayZ.

**Возможности:**
- Табличное редактирование
- Поиск по параметрам
- Массовое изменение значений
- Импорт и экспорт XML
- Работа с `cfglimitsdefinition.xml`

#### Формулы массового редактирования

| Формула | Описание | Пример |
|---------|----------|--------|
| `N` | Установить значение | `10` → установит 10 |
| `+N` | Прибавить значение | `+5` → увеличит на 5 |
| `-N` | Вычесть значение | `-3` → уменьшит на 3 |
| `*N` | Умножить на значение | `*2` → удвоит значение |
| `/N` | Разделить на значение | `/2` → уменьшит вдвое |
| `+N%` | Увеличить на процент | `+50%` → увеличит на 50% |
| `-N%` | Уменьшить на процент | `-25%` → уменьшит на 25% |

---

### 🔧 Recipe Editor
Визуальный редактор рецептов крафта.

**Возможности:**
- Создание рецептов через интерфейс

- Несколько вариантов результатов

- Импорт и экспорт `.c` файлов

## ❓ Часто задаваемые вопросы

### Как импортировать существующий types.xml в Types Editor?
1. Нажмите кнопку "Загрузить" в боковой панели Types Editor
2. Выберите ваш файл `types.xml`
3. Данные будут автоматически загружены в таблицу

### Как экспортировать рецепт из Recipe Editor?
1. Настройте рецепт в Recipe Editor
2. Нажмите "Сгенерировать код" в разделе "Код"
3. Нажмите "Сохранить" для экспорта в `.c` файл

### Что делать если данные не сохраняются?
Инструменты используют `localStorage` для автосохранения. Данные сохраняются автоматически при изменениях.

## 📊 Формулы массового редактирования (Types Editor)

| Формула | Описание | Пример |
|---------|----------|--------|
| `N` | Установить значение | `10` → установит 10 |
